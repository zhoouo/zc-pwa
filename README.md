# 🍃 ZC | 情侶王國

> **這不是一般的待辦清單，而是屬於你們兩個人的小宇宙。**

`ZC` 是一款專為情侶設計的 PWA（漸進式 Web 應用）。它將日常的互相照顧與承諾，轉化為一種有溫度、有紀錄、且具備儀式感的互動循環。

![ZC Banner](public/pwa-512.png)

## ✨ 核心理念

- **瑣事不瑣碎**：讓每一份付出都被看見，讓每一聲謝謝都有回音。
- **金幣作為語言**：建立專屬於兩人的交換機制，讓獎勵成為甜蜜的承諾。
- **克制與美感**：捨棄繁雜的社交與遊戲化元素，回歸安靜、私密且高級的視覺體驗。

---

## 🔄 核心循環

1.  **發起任務**：一方提出請求或約定。
2.  **努力達成**：另一方完成後送出審核。
3.  **看見付出**：發起方審核通過，發放金幣。
4.  **兌現承諾**：使用金幣在商城兌換對方提供的專屬獎勵。

---

## 🚀 功能亮點

- **首頁總覽**：一眼看清雙方金幣餘額與待辦事項。
- **任務系統**：完整的建立、送審、通過、退回流程。
- **專屬商城**：自定義承諾清單（如「按摩 30 分鐘」、「一頓晚餐」）。
- **金庫流水**：每一筆金幣變動都可追溯，記錄點滴付出。
- **隱藏驚喜**：內置隱藏商城入口，留給彼此的小暗號。
- **極致美學**：暖米色調、毛玻璃質感、優雅的細線條設計。
- **PWA 支援**：可安裝至手機主畫面，提供接近原生 App 的流暢體驗。

---

## 🛠️ 技術棧

- **前端**: [Vue 3](https://vuejs.org/) + [Vite](https://vitejs.dev/) + [TypeScript](https://www.typescriptlang.org/)
- **樣式**: [Tailwind CSS](https://tailwindcss.com/) (極簡優雅風格)
- **資料層**: [Supabase](https://supabase.com/) (Auth / Database / Realtime)
- **部署與 PWA**: [Vercel](https://vercel.com/) + `vite-plugin-pwa`

---

## 📦 快速開始

### 1. 環境要求

- Node.js (LTS 版本)
- Supabase 專案 (需配置 `.env`)

### 2. 配置環境變數

在根目錄建立 `.env` 文件：

```env
VITE_SUPABASE_URL=你的_SUPABASE_URL
VITE_SUPABASE_ANON_KEY=你的_SUPABASE_ANON_KEY
```

### 3. 安裝與運行

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev

# 構建生產版本
npm run build
```
